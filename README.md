# v-tinyosc

C [tinyosc](https://github.com/funatsufumiya/tinyosc) binding for V language 

> [!NOTE]
> If you need pure V implementation, use [vosc](https://github.com/funatsufumiya/vosc) instead.

## Install

```bash
$ git clone https://github.com/funatsufumiya/v-tinyosc ~/.vmodules/tinyosc
```

## Usage

see [sender example](./examples/example_sender/main.v) and [receiver example](./examples/example_receiver/main.v) code.
